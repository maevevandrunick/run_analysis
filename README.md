This R script was created to process the samsung data set and produce a tidy data set for the Getting and Cleaning Data assignment.
It follows these steps as outlined in the assignment brief 
1)Merges the training and test datasets into a single dataset.
2)Extracts only measurements representing means and standard deviations.
3)Replaces activity IDs with descriptive activity names.
4)Labels the dataset with descriptive variable names.
5) Creates a second tidy dataset containing the average of each variable for each activity and each subject.
